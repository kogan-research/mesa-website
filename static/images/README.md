# Images Directory

Place your pictogram images here. The images should be SVG or PNG format and represent the theme/area of each card.

## Recommended Image Specifications

- **Format**: SVG (preferred) or PNG
- **Size**: 200x200px to 400x400px
- **Style**: Simple, clean pictograms or icons
- **Color**: Will be automatically inverted to white on gradient backgrounds

## Image Paths Used in Content

- `/images/research-area-1.svg` - Research Area 1 pictogram
- `/images/research-area-2.svg` - Research Area 2 pictogram
- `/images/research-area-3.svg` - Research Area 3 pictogram
- `/images/project-1.svg` - Project 1 pictogram
- `/images/project-2.svg` - Project 2 pictogram
- `/images/publication-icon.svg` - Publication/news icon
- `/images/updates-icon.svg` - Lab updates icon
- `/images/events-icon.svg` - Events icon

- [The Noun Project](https://thenounproject.com/) - Icon library
- [Heroicons](https://heroicons.com/) - Free SVG icons

Replace the placeholder paths in your content files with your actual image paths.
